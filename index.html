Parrafos

Aqui voy a mostrar los ejmplos basicos de una pagina creada en html. <p>Este es un párrafo en HTML</p>

Encabezados

<h1> </h1>

<h2> </h2>

<h3> </h3>

<h4> </h4>

<h5> </h5>
<h6> </h6>

Lo anterior son encabezados se escriben con<h1> </h1> el numero simboliza el tamaño va de 1 a 6 a mas mayor el numero mas pequeño el encabezado

Tipos de texto

<b>El texto en negrita se escribe asi</b>

<u>El texto en subrayado se escribe asi</u>

<i>El texto en cursiva se escribe asi</i>

Tambien se pueden mezclar todos

Listas

Existen dos tipos de listas <ol>ordenadas</ol> y <ul>desordenadas</ul> los elementos de las listas se escriben entre <li></li>

  1. Esto es
  2. una lista
  3. Ordenada
  • Esto es
  • una lista
  • Desordenada

Imagenes

Para introducir una imagen se escibe <img src ="imagen.png" alt="descripción">

La ruta de la imagen se describe en el src pudiendo ser un directorio o un link y tambien podemos añadir otros datos como la altura y anchura de la imgane con width="300" height="300"
